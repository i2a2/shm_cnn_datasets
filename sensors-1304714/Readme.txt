This dataset corresponds to the article titled "Damage Localization in Composite Plates Using Wavelet Transform and 2-D Convolutional Neural Networks"
Journal: MDPI Sensors
doi: https://doi.org/10.3390/s21175825

Data is arranged in matlab .mat files containing data [time_samples,transmitter,receiver] and feat (features) variables.
Each file corresponds to a single Round-Robin/Full Matrix Capture Test.
The sampling rate at which data was acquired is 12.5 MHz.
